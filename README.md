# IndusMind AI - Industrial Knowledge Intelligence

AI-powered platform for P&ID OCR, Knowledge Graph Analysis, and Predictive Maintenance.

## 🚀 Deployment
1. **Frontend**: Deploy `index.html` to Vercel.
2. **Databases**: Run `docker-compose up -d`.
3. **Backend**: Run `pip install -r requirements.txt` and `python backend_engine.py`.

## 🛠 Features
- **Universal OCR**: PDF/Image technical document extraction.
- **Maintenance RCA**: Root cause analysis using Gemini 1.5 Pro.
- **Knowledge Graph**: Relationship mapping for industrial assets.
